## Features
- Video Call Conference
- One on One Video Call and Gallery Layout
- Switch Camera
- Switch Audio Output
    
## Tech Stack
**Client:** Flutter, Zego UI Kit

**Server:** ZegoCloud

## Run Locally
Clone the project

```bash
  git clone https://github.com/fanolans/face-room.git
```

Go to the project directory

```bash
  cd project-folder
```

Install dependencies

```bash
  flutter pub get
```

Start the app

```bash
  flutter run
```
